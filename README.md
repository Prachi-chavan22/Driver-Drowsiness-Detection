# Driver Drowsiness Detection System

A multimodal driver drowsiness detection system using Computer Vision and Voice Interaction.

## Features
- Eye drowsiness detection (EAR)
- Yawning detection (MAR)
- Head pose detection
- Voice alert system
- Speech-based driver interaction

## Technologies Used
- Python
- OpenCV
- MediaPipe
- NumPy
- SciPy
- pyttsx3
- SpeechRecognition
